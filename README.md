# JavaScript_Code-BitShiftingSQLInjection

## 왜 개발 하였나?
Blind SQL Injection이란 아무것도 모르는 상태에서 하나 하나 질문을 하고 참 & 거짓이 반환되는 값으로 정보를 탈취하는 공격 기법이다. 모의해킹을 진단 하던 중 일일히 입력 및 요청 해야하는 공격을 자동으로 요청하여 원하는 값을 획득하기 위하여 개발했습니다.
